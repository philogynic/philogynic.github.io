RNN and LSTM forward propagationÂ¶

Let's try the example. Try to do forward propagation 1 time using RNN and LSTM model for full input sequence.

  1. RNN using nn.Linear
  2. RNN using nn.RNN
    1. with for loop (loop sequence of input element one at a time)
    2. entire sequence at once
  3. same as (2) but for LSTM using nn.LSTM

Beside doing forwad propagation, I also going to demonstrate how to use loss function and wheter we can succesfully use backward propagation to that loss function or not. To be able to to this is of course very important

class RNN_1(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(RNN_1, self).__init__()
        self.hidden_size = hidden_size
        self.i2h = nn.Linear(input_size + hidden_size, hidden_size)
        self.i2o = nn.Linear(input_size + hidden_size, output_size) 
        self.softmax = nn.LogSoftmax()
        
    def forward(self, input, hidden):
        combined = torch.cat((input, hidden), 1)
        hidden = self.i2h(combined)
        output = self.i2o(combined)
        output = self.softmax(output)
        return output, hidden
    
    def initHidden(self):
        return Variable(torch.zeros(1, self.hidden_size))

2. A. RNN using nn.RNN with for loop one element at a timeÂ¶

After constructing RNN manually using Linear model, some people get confused how to make the same model by utilizing the nn.RNN module. Yes I think it can be quite confusing since nn.Linear and nn.RNN is different by nature. Module nn.RNN by default already be able to construct for loop by itself internally. But we can also force it to behave as previous model by manually constructing for loop.

http://pytorch.org/docs/master/nn.html#rnn

class torch.nn.RNN(*args, **kwargs)

Applies a multi-layer Elman RNN with tanh or ReLU non-linearity to an input sequence. For each element in input sequnce.

Parameters:

  • input_size: the number of expected features in the input x
  • hidden_size: the number of features in the hidden state
  • num_layers: number of recurrent layers.
  • how about the output dude?

Input: input, h0

  • input: tensor containing the features of the input sequence
  • h0: tensor containing the initial hidden state

Output: output, hn

  • output: tensor containing the output features from the last layer
  • hn: tensor containing the hidden state

class RNN_2A(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(RNN_2A, self).__init__()
        self.hidden_size = hidden_size
        self.rnn = nn.RNN(input_size, hidden_size)
        self.h2o = nn.Linear(hidden_size, output_size)
        self.softmax = nn.LogSoftmax()
        
    def forward(self, input, h0):
        output, hn = self.rnn(input, h0)
        output = self.h2o(output.view(1,-1))
        output = self.softmax(output)
        return output, hn
    
    def initHidden(self):
        return Variable(torch.zeros(1, 1, self.hidden_size))

class RNN_2B(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(RNN_2B, self).__init__()
        self.hidden_size = hidden_size
        self.rnn = nn.RNN(input_size, hidden_size)
        self.h2o = nn.Linear(hidden_size, output_size)
        self.softmax = nn.LogSoftmax()
        
    def forward(self, input, h0):
        rnn_out, hn = self.rnn(input, h0)
        lin_out = self.h2o(rnn_out.view(len(input),-1))
        output = self.softmax(lin_out)
        return output, hn
    
    def initHidden(self):
        return Variable(torch.zeros(1, 1, self.hidden_size))

3. A. LSTM using nn.LSTM with for loop one element at a timeÂ¶

Just like nn.RNN, nn.LSTM also has internal foor loop since they both designed to do sequence model. But now as we did before in RNN, let's construct the for loop manually.

Inputs: input, (h0, c0)

  • input: input features
  • h0: initial hidden state
  • c0: initial cell state

Output: output, (hn, cn)

  • output: output features
  • hn: hidden state
  • cn: cell state

class LSTM_A(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(LSTM_A, self).__init__()
        self.hidden_size = hidden_size
        self.lstm = nn.LSTM(input_size, hidden_size)
        self.h2o = nn.Linear(hidden_size, output_size)
        self.softmax = nn.LogSoftmax()
        
    def forward(self, input, hidden):
        lstm_out, hidden = self.lstm(input, hidden)
        output = self.h2o(lstm_out.view(1,-1))
        output = self.softmax(output)
        return output, hidden
    
    def initHidden(self):
        h0 = Variable(torch.zeros(1, 1, self.hidden_size))
        c0 = Variable(torch.zeros(1, 1, self.hidden_size))
        return (h0, c0)

class LSTM_B(nn.Module):
    def __init__(self, input_size, hidden_size, output_size):
        super(LSTM_B, self).__init__()
        self.hidden_size = hidden_size
        self.lstm = nn.LSTM(input_size, hidden_size)
        self.h2o = nn.Linear(hidden_size, output_size)
        self.softmax = nn.LogSoftmax()
        
    def forward(self, input, h0):
        lstm_out, hn = self.lstm(input, h0)
        lin_out = self.h2o(lstm_out.view(len(input),-1))
        output = self.softmax(lin_out)
        return output, hn
    
    def initHidden(self):
        h0 = Variable(torch.zeros(1, 1, self.hidden_size))
        c0 = Variable(torch.zeros(1, 1, self.hidden_size))
        return (h0,c0)
